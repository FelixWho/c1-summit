# c1-summit

## How to Setup

1. Setup Virtual Environment

```bash
virtualenv env
```

2. Activate virtual environment

```bash
source env/bin/activate
```

3. Install requirements

```bash
pip3 install -r requirements
```

4. Start Flask app

```bash
source .env
flask run
```

--- or ---

4. Start React app

```bash
cd public
npm start
```
